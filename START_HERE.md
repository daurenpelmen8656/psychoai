## Что нужно сделать ВАМ 👇

### Шаг 1️⃣: Откройте команду
```
Клавиши: Windows + R
Введите: cmd
Нажмите: Enter
```

### Шаг 2️⃣: Перейдите в папку проекта
```bash
cd c:\Users\boyt6\Documents\GitHub\psychoai
```

### Шаг 3️⃣: Установите зависимости (если не установлены)
```bash
npm install
```
(Подождите, пока завершится - может занять несколько минут)

### Шаг 4️⃣: Запустите приложение
```bash
npm start
```

### Шаг 5️⃣: Браузер откроется автоматически
Если нет - откройте вручную: `http://localhost:3000`
