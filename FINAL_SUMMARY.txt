                        📊 ТАБЛИЦЫ И ИНТЕГРАЦИЯ
================================================================================

ТАБЛИЦА: users
  ├─ id (уникальный ID)
  ├─ name (имя пользователя)
  ├─ language (выбранный язык)
  ├─ goal (цель)
  ├─ therapist (AI психолог)
  ├─ created_at (когда зарегистрировался)
  └─ updated_at (последнее обновление)

  💾 СОХРАНЯЕТСЯ: Когда пользователь заполняет Onboarding форму
  🔄 ЗАГРУЖАЕТСЯ: При логине (не реализовано, но таблица есть)

ТАБЛИЦА: messages
  ├─ id (уникальный ID)
  ├─ user_id (ID пользователя)
  ├─ role ('user' или 'ai')
  ├─ content (текст сообщения)
  └─ created_at (когда отправлено)

  💾 СОХРАНЯЕТСЯ: Каждое сообщение в чате (user и ai)
  🔄 ЗАГРУЖАЕТСЯ: Вся история при открытии Chat

ТАБЛИЦА: chat_sessions (опционально)
  ├─ id
  ├─ user_id
  ├─ title
  ├─ is_active
  ├─ created_at
  └─ updated_at

ТАБЛИЦА: user_profiles (опционально)
  ├─ id
  ├─ user_id
  ├─ bio
  ├─ avatar_url
  ├─ preferences (JSONB)
  ├─ created_at
  └─ updated_at
