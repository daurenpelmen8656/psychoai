## 🚀 ЧТО ДЕЛАТЬ ДАЛЬШЕ (3 ПРОСТЫХ ШАГА)

### Шаг 1️⃣: Создать таблицы в Supabase

**ОТКРОЙ:** https://app.supabase.com/

**ВЫПОЛНИ:**
1. Выбери свой проект
2. Слева нажми "SQL Editor"
3. Нажми "New Query"
4. Открой файл `supabase.sql` в этом проекте
5. Скопируй ВСЁ содержимое (Ctrl+A → Ctrl+C)
6. Вставь в Supabase SQL Editor (Ctrl+V)
7. Нажми "Run" или Ctrl+Enter
8. Готово! ✅

### Шаг 2️⃣: Запустить приложение

```bash
npm start
```

### Шаг 3️⃣: Проверить что работает

1. **Откроется браузер на http://localhost:3000**
2. **Нажми "Get Started Free"**
3. **Заполни форму регистрации:**
   - Имя: например "Иван"
   - Язык: English или Русский
   - Цель: выбери любую
   - AI психолог: выбери любого
4. **Нажми "Start"**
5. **Напиши сообщение в чате**
6. **Проверь Supabase:**
   - Открой https://app.supabase.com/
   - Выбери проект
   - Table Editor слева
   - Выбери таблицу "users" - твои данные должны быть там!
   - Выбери таблицу "messages" - твои сообщения должны быть там!

## 📊 Структура таблиц

### Таблица `users`
```
id              - уникальный ID пользователя
name            - имя пользователя
language        - выбранный язык
goal            - выбранная цель
therapist       - выбранный AI психолог
created_at      - когда зарегистрировался
updated_at      - когда последний раз обновлен
```

### Таблица `messages`
```
id              - уникальный ID сообщения
user_id         - ID пользователя
role            - 'user' или 'ai'
content         - текст сообщения
created_at      - когда отправлено
```

## 🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Таблицы не видны в Supabase
- Проверь что SQL скрипт выполнился без ошибок
- Открой Table Editor - должны быть таблицы users, messages и т.д.

### Данные не сохраняются при регистрации
- Открой консоль браузера (F12 → Console)
- Смотри есть ли ошибки
- Проверь что таблица `users` существует в Supabase

### История чата не загружается
- Проверь консоль браузера на ошибки
- Убедись что таблица `messages` существует
- Проверь что сообщения сохраняются в Supabase

### "Table users does not exist"
- Ты не выполнил SQL скрипт
- Открой supabase.sql, скопируй код
- Выполни в Supabase SQL Editor
