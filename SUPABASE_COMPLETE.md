## 🚀 ГОТОВАЯ ПОСЛЕДОВАТЕЛЬНОСТЬ (3 ШАГА)

### ШАГ 1: СОЗДАТЬ ТАБЛИЦЫ (1 минута)

```
1. Откройте: https://app.supabase.com/
2. Выберите проект
3. SQL Editor → New Query
4. Откройте файл: supabase.sql
5. Скопируйте ВСЕ содержимое (Ctrl+A)
6. Вставьте в редактор (Ctrl+V)
7. Нажмите: Run или Ctrl+Enter
8. Готово! ✅
```

### ШАГ 2: ЗАПУСТИТЬ ПРИЛОЖЕНИЕ (30 секунд)

```bash
npm start
```

Откроется браузер на http://localhost:3000

### ШАГ 3: ПРОВЕРИТЬ РАБОТАЕТ (2 минуты)

```
1. Нажмите: "Get Started Free"
2. Заполните форму регистрации
3. Нажмите: "Start"
4. Напишите сообщение в Chat
5. Откройте Supabase Dashboard
6. Table Editor → смотрите таблицы
7. Ваши данные должны быть там! 🎉
```

## 🆘 ПРОБЛЕМЫ И РЕШЕНИЯ

### "Table does not exist"
→ Выполни SQL скрипт в Supabase (supabase.sql)

### "Data doesn't save"
→ Проверь F12 Console на ошибки, смотри Supabase таблицы

### "Messages don't load"
→ Проверь что таблица `messages` существует, проверь user_id

### "Permission denied"
→ Включи RLS policies (должны быть в SQL скрипте)

### "Env variables not working"
→ Перезагрузи приложение (npm start заново)
